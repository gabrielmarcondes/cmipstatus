Então vamos falar de tecnologia.

Este site se baseou principalmente em dois projetos:


Django

O Django é um framework Python para web e é a base de todo esse site

Se você já sabe Python e quer começar a programar com web, eu recomendo muito fortemente. A documentação é excelente e há uma comunidade grande e prestativa. Praticamente tudo que você precisar você vai achar pronto por aí: é só plugar e usar.

Bootstrap

O Bootstrap é uma coleção de CSSs e JavaScripts e é o responsável por este site ser bonito

Foi criado pelo pessoal do Twitter para facilitar a vida deles lá dentro, e depois disponibilizado para qualquer desenvolvedor usar no seu site.

Este site usa também alguns ícones dos Glyphicons, que vieram embutidos no Bootstrap
